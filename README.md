# Olga Waldschmidt Portfolio

Modern React rewrite of the original exported static homepage.

## Stack

- React + Vite
- Tailwind CSS v4
- ESLint (scoped to `src/`)

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Assets and Domain

- Portfolio images are served from `public/images`.
- Custom domain is preserved via `public/CNAME`.

## Notes

- The legacy exported files (`Startseite.html`, `nicepage.css`, etc.) are kept in the repository as reference material.
- The active app entry is `src/App.jsx`.
